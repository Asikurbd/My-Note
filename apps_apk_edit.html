Android APK-তে লোগো পরিবর্তন করার ধাপগুলো নিচে দেওয়া হলো:


লোগো পরিবর্তনের ধাপ:


# ১. প্রস্তুতি:
- APK Editor App (Android)
- নতুন লোগো ইমেজ (সঠিক রেজোলিউশনে)
- File Manager App

# ৩. লোগো ফাইলের নাম চিহ্নিত করা:
সাধারণ নামগুলো:
- ic_launcher.png

- ic_launcher_background.png

# ৪. APK Editor দিয়ে পরিবর্তন করা:
ধাপ-১: APK Editor ওপেন করুন
ধাপ-২: "Select an APK file" এ ক্লিক করুন
ধাপ-৩: আপনার APK সিলেক্ট করুন
ধাপ-৪: "Full Edit" মোড নির্বাচন করুন
ধাপ-৫: উপরের পাথগুলোতে গিয়ে লোগো ফাইল খুঁজুন
ধাপ-৬: পুরোনো লোগো ডিলিট করে নতুন লোগো অ্যাড করুন
ধাপ-৭: Save/Install করুন


# ৫. লোগো রেজোলিউশন গুরুত্বপূর্ণ:
| ফোল্ডার | রেজোলিউশন |
|---------|------------|
| mipmap-mdpi | 48x48 px |
| mipmap-hdpi | 72x72 px |
| mipmap-xhdpi | 96x96 px |
| mipmap-xxhdpi | 144x144 px |
| mipmap-xxxhdpi | 192x192 px |


- সব রেজোলিউশনে লোগো replace করুন
- ফাইল ফরমেট PNG হওয়া ভালো
- ফাইলের নাম একই রাখুন
- Backup নিয়ে কাজ করুন
